Welcome to Book Manager

Book Manager is a tool designed to help you manage your home library. With Book Manager, you can:

  • Add, remove, and edit books in your collection
  • Search for books
  • Each user has their own personal database of books

Additional Features

Book Management

Easily manage your book collection by adding, removing, or editing books.

Search Functionality

Quickly find books in your collection using the search feature.

Personalized Database

Each user has their own personalized database of books.
